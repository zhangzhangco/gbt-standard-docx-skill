2. 基础规则校验

1. 提供公开可运行的最小模板方案或模板接入说明。

3. 把 README 从开发者视角继续收敛成使用者视角。

## 第三优先级

1. 强化目录刷新与复杂交叉引用回写。
2. 增加更多版式级审查项。
3. 把审查 JSON 输出进一步固化成正式报告模板。
